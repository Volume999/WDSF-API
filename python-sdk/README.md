# Python SDK for WDSF API

This SDK allows querying WDSF api, powered by strong typing and clear communication interface

## Development

Install dependencies
```bash
  poetry install
```

Activate shell
```bash
poetry shell
# Or
eval $(poetry env activate)
```

Run formatting
```bash
make fmt
```

Run style check & tests
```bash
make
```
